# **App Name**: Color Dash Blitz

## Core Features:

- Quick Match Gameplay: Present the user with a main color and several choices. The user must quickly select the matching color or complete a simple color-related puzzle.
- Timed Challenge Mode: A mode where players aim to achieve the highest score within a set time limit, increasing difficulty as they progress.
- Dynamic Score Tracking: Display real-time scores during gameplay and a final score summary after each round.
- Difficulty Progression: Gradually increase the complexity of color choices or patterns as the player successfully completes rounds.
- Yandex Games Leaderboard Integration: Submit player high scores to Yandex Games leaderboards for competitive play. (Note: using xsoulspace_ysdk_games_js dart:js_interop library)
- Yandex Games Ad Monetization: Integrate Yandex Games ad APIs to display interstitial or rewarded video ads.
- AI-Powered Color Fact Tool: Provide a fun, randomly generated fact about the current game's primary color or a general color theory tidbit upon game completion or during loading screens.

## Style Guidelines:

- Primary color: Energetic coral (#FF5E4C) to evoke excitement and speed, aligning with the hyper-casual, quick-reaction gameplay.
- Background color: A very light, warm peach (#FFF4F3) providing a soft contrast that doesn't distract from the vibrant gameplay elements.
- Accent color: A vibrant fuchsia (#E02B89) for interactive elements, highlights, and subtle call-to-action indicators, complementing the primary coral with good contrast.
- Font: 'Poppins' (sans-serif) for both headlines and body text. Its geometric and contemporary feel is ideal for a modern, playful, and clean hyper-casual game interface.
- Use minimalist, outline-style icons with rounded edges for all interactive elements (play button, settings, leaderboard), ensuring clarity and a friendly appearance.
- Prioritize a clean, uncluttered layout with central alignment for core game elements, ensuring quick comprehension and accessibility for rapid decision-making.
- Incorporate subtle, fast-paced feedback animations, such as a quick bounce or glow effect upon correct selection and a short shake on incorrect answers, to enhance player engagement without slowing down gameplay.
